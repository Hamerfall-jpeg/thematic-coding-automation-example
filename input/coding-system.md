The following codes are organized by the biopsychosocial model domains. Each code includes a unique identifier, name in Dutch and English, definition, and criteria for when to apply or not apply the code.

## Domain 1: Biological/Physical Dimension

| BIO-01 | Lage rugpijn - Ervaring / Low back pain - Experience |
| --- | --- |
| Definition | Description of the physical pain experience, intensity, location, quality |
| When to Apply | When participant describes their pain sensations, severity, or physical symptoms |
| When NOT to Apply | When discussing causes or diagnosis only |
| Example | "De pijn zit diep in mijn onderrug, het voelt als een constante druk" |

| BIO-02 | Lage rugpijn - Impact op functie / LBP - Functional impact |
| --- | --- |
| Definition | How pain affects physical activities and daily functioning |
| When to Apply | When participant describes limitations in movement, ADL, or physical tasks |
| When NOT to Apply | When discussing emotional impact only |
| Example | "Ik kan niet meer dan 10 minuten staan zonder dat de pijn erger wordt" |

| BIO-03 | Fysieke zelfmanagement strategieën / Physical self-management strategies |
| --- | --- |
| Definition | Physical activities, exercises, or movements used to manage pain |
| When to Apply | When participant describes specific physical actions they take (exercises, stretching, posture, activity modification) |
| When NOT to Apply | When describing what physiotherapist does TO them (passive treatment) |
| Example | "Ik doe elke ochtend de rugstabilisatie oefeningen" |

| BIO-04 | Begrip van fysieke aspecten / Understanding physical aspects |
| --- | --- |
| Definition | Patient's understanding of the physical/biological aspects of their condition |
| When to Apply | When participant explains their understanding of what's happening physically |
| When NOT to Apply | When repeating what therapist said without understanding |
| Example | "Ik begrijp nu dat mijn spieren zwak zijn geworden" |


## Domain 2: Psychological Dimension

| PSY-01 | Emotionele impact / Emotional impact |
| --- | --- |
| Definition | Emotional responses to chronic pain (frustration, fear, anxiety, depression) |
| When to Apply | When participant describes feelings, emotions, or mood related to pain |
| When NOT to Apply | When describing physical sensations only |
| Example | "Ik voel me gefrustreerd omdat ik niet meer kan doen wat ik wil" |

| PSY-02 | Angst-vermijding / Fear-avoidance |
| --- | --- |
| Definition | Fear of movement or activities that might cause pain; avoidance behaviors |
| When to Apply | When participant expresses fear of specific movements or activities, or avoids activities due to fear |
| When NOT to Apply | When avoiding due to actual physical inability rather than fear |
| Example | "Ik durf niet te bukken, bang dat het weer uitschiet" |

| PSY-03 | Zelfvertrouwen / Self-efficacy |
| --- | --- |
| Definition | Confidence in ability to manage pain or perform activities |
| When to Apply | When participant expresses confidence or lack thereof in managing symptoms |
| When NOT to Apply | When discussing actual abilities rather than confidence |
| Example | "Ik weet nu dat ik het zelf in de hand heb" |

| PSY-04 | Psychologische zelfmanagement / Psychological self-management |
| --- | --- |
| Definition | Mental strategies used to cope (e.g., pacing, goal-setting, cognitive strategies) |
| When to Apply | When participant describes mental/cognitive strategies for coping |
| When NOT to Apply | When describing physical strategies only |
| Example | "Ik probeer mijn dag in kleine stukjes te verdelen" |

| PSY-05 | Acceptatie / Acceptance |
| --- | --- |
| Definition | Acceptance or non-acceptance of chronic pain as part of life |
| When to Apply | When participant discusses coming to terms with or resisting their chronic condition |
| When NOT to Apply | When discussing diagnosis understanding only |
| Example | "Ik moet leren leven met deze pijn" |


## Domain 3: Social Dimension

| SOC-01 | Sociale context (+ effecten) / Social context (+ effects) |
| --- | --- |
| Definition | Impact of pain on social life, relationships, social activities |
| When to Apply | When participant describes how pain affects their social interactions, relationships, or participation |
| When NOT to Apply | When discussing work only (use SOC-02) |
| Example | "Ik ga niet meer naar verjaardagen omdat ik niet lang kan zitten" |

| SOC-02 | Werk impact / Work impact |
| --- | --- |
| Definition | Effect of pain on work, occupational functioning, or work-related activities |
| When to Apply | When participant discusses work capacity, work limitations, or work-related challenges |
| When NOT to Apply | When discussing leisure activities |
| Example | "Ik werk nu in deeltijd omdat ik het niet meer volhoud" |

| SOC-03 | Sociale steun / Social support |
| --- | --- |
| Definition | Support (or lack thereof) from family, friends, colleagues, or healthcare providers |
| When to Apply | When participant describes receiving or lacking support from others |
| When NOT to Apply | When discussing treatment only |
| Example | "Mijn vrouw helpt me met dingen die ik niet meer kan" |

| SOC-04 | Sociale verwachtingen / Social expectations |
| --- | --- |
| Definition | Expectations or pressure from others; how others view their condition |
| When to Apply | When participant discusses what others expect of them or how others perceive their pain |
| When NOT to Apply | When discussing their own expectations |
| Example | "Mensen begrijpen niet dat ik pijn heb als ze me normaal zien bewegen" |


## Domain 4: Biopsychosocial Integration

| BPS-01 | Begrip biopsychosociaal model / Understanding biopsychosocial model |
| --- | --- |
| Definition | Patient's understanding of how bio-psycho-social factors interact in their pain |
| When to Apply | When participant explicitly connects multiple domains or expresses holistic understanding |
| When NOT to Apply | When discussing only one domain |
| Example | "Ik merk dat stress mijn pijn verergert en dan kan ik minder doen" |

| BPS-02 | Holistische benadering ervaring / Holistic approach experience |
| --- | --- |
| Definition | Experience of receiving (or not receiving) biopsychosocial care from physiotherapist |
| When to Apply | When participant describes how physiotherapist addresses multiple dimensions |
| When NOT to Apply | When describing purely physical treatment |
| Example | "Mijn fysiotherapeut vraagt ook naar hoe het met me gaat" |


## Domain 5: Self-Management Process

| SM-01 | Zelfmanagement - Kennis / Self-management - Knowledge |
| --- | --- |
| Definition | Knowledge about self-management strategies and condition |
| When to Apply | When participant demonstrates or discusses knowledge about managing their condition |
| When NOT to Apply | When repeating information without understanding |
| Example | "Ik weet dat bewegen belangrijk is, ook als het pijn doet" |

| SM-02 | Zelfmanagement - Vaardigheden / Self-management - Skills |
| --- | --- |
| Definition | Practical skills for self-management (e.g., exercise technique, pacing, problem-solving) |
| When to Apply | When participant describes specific skills they've learned or apply |
| When NOT to Apply | When describing knowledge only, not application |
| Example | "Ik kan nu zelf voelen wanneer ik moet pauzeren" |

| SM-03 | Zelfmanagement - Zelfvertrouwen / Self-management - Confidence |
| --- | --- |
| Definition | Confidence in self-managing the condition |
| When to Apply | When participant expresses confidence or lack thereof specifically regarding self-management |
| When NOT to Apply | When discussing general self-efficacy |
| Example | "Ik voel me nu zekerder in het omgaan met mijn klachten" |

| SM-04 | Zelfmanagement - Zelfstandigheid / Self-management - Autonomy |
| --- | --- |
| Definition | Sense of independence and autonomy in managing condition |
| When to Apply | When participant expresses feeling in control or dependent on others |
| When NOT to Apply | When discussing actual abilities only |
| Example | "Ik ben niet meer afhankelijk van de therapeut" |

| SM-05 | Zelfmanagement - Barrières / Self-management - Barriers |
| --- | --- |
| Definition | Obstacles or challenges to self-management |
| When to Apply | When participant describes difficulties, obstacles, or challenges in self-managing |
| When NOT to Apply | When describing pain symptoms only |
| Example | "Ik heb geen tijd om de oefeningen elke dag te doen" |

| SM-06 | Zelfmanagement - Facilitators / Self-management - Facilitators |
| --- | --- |
| Definition | Factors that help or enable self-management |
| When to Apply | When participant describes what helps them self-manage effectively |
| When NOT to Apply | When describing treatment by therapist |
| Example | "Het helpt dat de oefeningen simpel zijn om thuis te doen" |


## Domain 6: Healthcare Relationship

| HC-01 | Therapeutische relatie / Therapeutic relationship |
| --- | --- |
| Definition | Quality of relationship with physiotherapist |
| When to Apply | When participant describes their relationship, communication, or interaction with physiotherapist |
| When NOT to Apply | When describing treatment techniques only |
| Example | "Mijn fysio luistert echt naar wat ik te zeggen heb" |

| HC-02 | Communicatie / Communication |
| --- | --- |
| Definition | Quality and clarity of communication with healthcare providers |
| When to Apply | When participant discusses how information is shared, explained, or understood |
| When NOT to Apply | When describing actual content of information |
| Example | "Hij legt alles goed uit zodat ik het begrijp" |

| HC-03 | Gedeelde besluitvorming / Shared decision-making |
| --- | --- |
| Definition | Involvement in treatment decisions and goal-setting |
| When to Apply | When participant describes being involved (or not) in making decisions about their care |
| When NOT to Apply | When following instructions only |
| Example | "Samen hebben we besloten welke oefeningen het beste passen" |

| HC-04 | Verwachtingen van zorg / Care expectations |
| --- | --- |
| Definition | What patient expects or hoped for from physiotherapy |
| When to Apply | When participant discusses what they wanted/expected from treatment |
| When NOT to Apply | When describing actual treatment received |
| Example | "Ik hoopte dat de fysio de pijn weg kon maken" |

| HC-05 | Tevredenheid met zorg / Care satisfaction |
| --- | --- |
| Definition | Satisfaction or dissatisfaction with physiotherapy care |
| When to Apply | When participant expresses satisfaction level with care received |
| When NOT to Apply | When describing specific treatment elements |
| Example | "Ik ben tevreden met de begeleiding die ik krijg" |


## Domain 7: Temporal Aspects

| TEMP-01 | Verloop tijd / Progress over time |
| --- | --- |
| Definition | Changes in pain, function, or self-management over time |
| When to Apply | When participant describes improvements, deterioration, or changes over time |
| When NOT to Apply | When describing current state only |
| Example | "In het begin kon ik helemaal niets, nu gaat het beter" |

| TEMP-02 | Toekomstperspectief / Future perspective |
| --- | --- |
| Definition | Expectations, hopes, or concerns about the future |
| When to Apply | When participant discusses future outlook, goals, or expectations |
| When NOT to Apply | When discussing past or present only |
| Example | "Ik hoop dat ik over een jaar weer kan sporten" |
